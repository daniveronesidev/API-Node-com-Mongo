# API Node com MongoDB 🚀

## 📌 Sobre o projeto
API desenvolvida com Node.js utilizando MongoDB como banco de dados.

## 🧱 Estrutura
- backend → lógica da aplicação
- frontend → interface
- db → conexão com banco
- models → schemas

## 🚀 Tecnologias
- Node.js
- Express
- MongoDB

## ▶️ Como rodar
```bash
npm install
